# php_formatter

## 目录说明

```
php_formatter
├── code  # 存放需要美化的php文件目录
├── vendor # 存放美化核心库目录
```

## 使用方法

### 将本项目克隆到本地

```sh
git clone https://github.com/zihelyu/php_formatter
```

### 进入php_formatter目录

```sh
cd php_formatter
```

### 使用php cli或网页访问方式运行

```sh
php format.php #php cli运行
http://localhost/php_formatter/format.php #网页访问运行
```
